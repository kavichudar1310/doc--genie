# doc--genie
python function documentation string
Doc-Genie is a project idea or tool that automatically generates documentation for Python functions.

When you write a Python function, Doc-Genie can automatically extract:

Function name

Parameters

Return values

Docstring (description written inside the function)

This helps developers save time and maintain clean documentation.
